Style Guide for Maths Tutoring Website

The color scheme

Light Blue

hsl(190, 58%, 66%)

This will be used to highlight the questions.

Light Green

hsl(118, 98%, 80%)

This will be used to highlight the worked answers.

Orange

hsl(30, 98%, 63%)

This will be used to highlight any extra details or aside boxes.

Bright Red

hsl(358, 100%, 49%)

This will be used to highlight any very important methods or theorems that MUST be remembered!

Fonts

Cambria (in italic)

This is the font that will be used for maths questions and equations.

Josefin Sans

This is the font that will be used for headings and titles

Xanh Mono

This is the font that will be used in the main block of writing.

Text Styles
